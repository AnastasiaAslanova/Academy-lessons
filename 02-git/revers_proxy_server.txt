Проски сервер представляет собой сетевой интерфейс связи, который обеспечивает коммуникацию между двумя удалёнными машинами.
Сервер получает данные, а затем перенаправляет их фактическому получателю.
Если целевая система, например веб сервера должна быть защищена вышестоящим прокси-сервером это называется обратный прокси сервер.
Обратный прокси сервер принимает запросы от внешних клиентовот имени серверов, в отличие от прямого прокси-сервера, кторый принимает запросы непосредственно от клиентов.
Обычный прокси- сервер скрывает идентификаторы клиентов, обратный скрывает идентификаторы серверов.
